# Spring-boot-rest application

Simple REST application using spring-boot and vue-js.


## Backend 
* Builder: Gradle
* PostgreSQL
* Java Persistence API
* Seurity
* OAuth2
* Thymeleaf
* Websocket
* Lombok

## Frontend
* Builder: Webpack 
* Babel
* Vue
* Vuetify
* Vuex
* Stomp
* Sockjs-client

## Database Managment System
* PostgreSQL 9.6-alpine ( docker )
